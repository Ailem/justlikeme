JustLikeMe
Life-insurance coverage benchmarking — see how your coverage compares to people like you, grounded in industry-standard actuarial methodology.

Live: https://getjustlikeme.com
What's here
index.html — the full site, a single self-contained static file (HTML, CSS, and JS inline). No build step required.
How it deploys
This repository is connected to Vercel. Every push to the default branch automatically builds and deploys to production at getjustlikeme.com — no manual upload needed.
Making changes
Edit index.html (or replace it with an updated version), commit, and push. Vercel handles the rest.
